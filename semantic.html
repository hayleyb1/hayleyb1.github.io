    Section One

    That heading is within section and article elements

    An aside element is used to mark additional information that can enhance another element but isn't required in order to understand the main content. Can be used along with sections or articles.

    Section Two

    Figure is used to encapsulate media such as an image. Figcaption is an element used to describe the media in the figure tags. Usually figcaption will go inside figure. This is different to using a p element to describe the content - if we decide to change the location of figure, the paragraph tag may get displaced from figure while a figcaption will move with the figure.
